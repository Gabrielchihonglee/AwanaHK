## AwanaHK for Awana Hong Kong

AwanaHK is a WordPress theme created for Awana Hong Kong. You can have a look at the live version at [Awana Hong Kong's Website](awana.org.hk).

### Framework or plugin used
- Bootstrap
- jQuery
- Font Awesome
